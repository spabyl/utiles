# About
A single header file for console colors for C++ projects

## Features:
- [x] UTF-8 support in Windows
- [x] Console Forecolors
- [ ] Console Backcolors
- [ ] Console title
- [ ] Cursor visibility
- [x] Clear screen
- [ ] LOG_INFO, LOG_WARN, LOG_ERROR macros
- [ ] timerstamp
